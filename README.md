## 🚀 Como Usar

### 1. Personalize os Dados

Edite o arquivo `index.html` e atualize:
- **Seu Nome**: Procure por "Guilherme Alves" e substitua
- **Seu Telefone**: Encontre "+5587999999999" (várias ocorrências)
- **Seu Email**: Substitua "seu-email@gmail.com"
- **Links Sociais**: LinkedIn, GitHub, Instagram, WhatsApp
- **Projetos**: Adicione seus próprios projetos na seção de portfólio

### 2. Personalize as Cores

No arquivo `style.css`, encontre a seção `:root` e altere:

```css
:root {
    --primary-color: #007AFF;      /* Azul principal */
    --primary-dark: #0051D5;       /* Azul escuro */
    --accent: #FF9500;             /* Cor destaque */
    /* ... outras cores */
}
```

### 3. Adicione Seus Projetos

Na seção de Portfolio (index.html), replique o card:

```html
<div class="portfolio-item">
    <div class="portfolio-image">
        <div class="portfolio-placeholder">Seu Projeto</div>
    </div>
    <div class="portfolio-info">
        <h3>Nome do Projeto</h3>
        <p>Descrição breve</p>
        <a href="link-do-projeto" class="view-project">Ver detalhes →</a>
    </div>
</div>
```

### 4. Configure Formulário de Contato

Para receber emails do formulário, você tem opções:
- **Formspree**: https://formspree.io/
- **Netlify Forms**: Se hospedar no Netlify
- **EmailJS**: Serviço gratuito para envio de emails
